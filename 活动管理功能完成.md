## 🚀 如何使用

### 1. 访问页面
1. 以管理员身份登录
2. 点击右上角用户菜单
3. 点击 "活动管理 / Event Management"
4. 或直接访问：`http://localhost:3000/zh/events`

### 2. 创建活动
1. 点击 "创建活动 / Create Event" 按钮
2. 填写表单：
   - **活动名称**（必填）：例如 "2025 华东区扯铃比赛"
   - **活动描述**（可选）：活动简介
   - **海报 URL**（可选）：海报图片链接
   - **背景视频 URL**（可选）：背景视频链接
   - **开始日期**（可选）
   - **结束日期**（可选）
3. 点击 "创建 / Create"
4. 看到成功提示："活动创建成功"

### 3. 编辑活动
1. 点击活动卡片上的编辑图标（铅笔）
2. 修改需要更改的字段
3. 点击 "更新 / Update"
4. 看到成功提示："活动更新成功"

### 4. 激活活动
1. 点击活动卡片上的 "激活 / Activate" 按钮
2. 该活动将被设置为激活状态
3. 其他活动自动变为非激活状态
4. 激活的活动会显示在评委登录页面

### 5. 删除活动
1. 点击活动卡片上的删除图标（垃圾桶）
2. 确认删除
3. 活动被永久删除

## 💡 使用提示

### 最佳实践
1. **海报图片**：建议使用 16:9 或 4:3 比例的图片
2. **背景视频**：建议使用 MP4 格式，文件大小不超过 50MB
3. **活动名称**：建议使用清晰、简洁的名称
4. **激活状态**：同一时间只能有一个活动处于激活状态
5. **删除操作**：删除前请确认，删除后无法恢复

### 常见问题

**Q: 如何更换评委登录页面的海报？**  
A: 编辑活动，更新海报 URL，然后激活该活动。

**Q: 可以同时激活多个活动吗？**  
A: 不可以，系统会自动将其他活动设为非激活状态。

**Q: 删除的活动可以恢复吗？**  
A: 不可以，删除是永久性的，请谨慎操作。

**Q: 海报和视频必须填写吗？**  
A: 不是必须的，只有活动名称是必填项。
