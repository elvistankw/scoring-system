## 📱 使用方法 / How to Use

### 评分流程 (Scoring Flow)

```
1️⃣ 选择选手
   ↓
2️⃣ 输入评分
   ↓
3️⃣ 自动保存 (300ms后)
   ↓
4️⃣ 看到 ✅ 已自动保存
   ↓
5️⃣ 继续下一个选手
```

## 🔍 常见问题 / FAQ

### Q1: 我需要点击保存按钮吗？
**A:** 不需要！评分会自动保存。

### Q2: 多久会自动保存？
**A:** 输入后 300 毫秒（0.3秒）自动保存。

### Q3: 如果我快速输入会怎样？
**A:** 系统会等你停止输入 300ms 后才保存，避免频繁保存。

### Q4: 刷新页面会丢失数据吗？
**A:** 不会！所有评分都缓存在本地，刷新后自动恢复。

### Q5: 如何知道评分已保存？
**A:** 看到绿色勾选图标 ✅ 和"已自动保存"文字。

### Q6: 可以修改已保存的评分吗？
**A:** 可以！修改后会自动重新保存。

### Q7: 如果我想清除评分怎么办？
**A:** 直接删除输入框中的数字，系统会自动更新状态。

## 🔧 故障排除 / Troubleshooting

### 问题1: 没有看到"已自动保存"
**解决方案:**
- 等待 300ms
- 检查是否输入了有效数字
- 刷新页面重试

### 问题2: 刷新后数据丢失
**解决方案:**
- 检查浏览器是否允许 localStorage
- 清除浏览器缓存后重试
- 确认在同一浏览器中操作

### 问题3: 保存状态一直显示"保存中"
**解决方案:**
- 检查输入的数据是否有效
- 刷新页面
- 联系技术支持
