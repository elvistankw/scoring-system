# 管理员评分管理 - 快速指南

## 🎯 功能位置
```
管理员控制台 → 比赛管理 → 选择比赛 → 评分管理标签
```

## 📋 功能列表

### ✅ 查看评分
- 选择选手查看其所有评分记录
- 显示评委姓名、提交时间
- 显示各项评分维度

### ✏️ 编辑评分
- 点击"编辑"按钮
- 修改评分值（0-30分）
- 保存更改

### 🗑️ 删除评分
- 点击"删除"按钮
- 确认删除
- 评分记录永久删除

## 🚀 快速操作

### 编辑评分
```
1. 选择选手
2. 点击评分记录的"编辑"图标 ✏️
3. 修改评分值
4. 点击"保存"
```

### 删除评分
```
1. 选择选手
2. 点击评分记录的"删除"图标 🗑️
3. 确认删除
```

## ⚠️ 注意事项

1. **权限要求**: 只有管理员可以使用此功能
2. **数据影响**: 修改/删除评分会影响选手排名
3. **缓存更新**: 修改后会自动清除缓存
4. **不可撤销**: 删除操作无法撤销，请谨慎操作

## 🔧 技术细节

### API 端点
- `PUT /api/scores/:id` - 更新评分
- `DELETE /api/scores/:id` - 删除评分

### 权限
- 需要管理员 JWT token
- 使用 `requireRole('admin')` 中间件

### 缓存
- 自动清除 Redis 缓存
- 确保数据一致性

## 📊 评分字段

| 比赛类型 | 评分维度 |
|---------|---------|
| 个人赛 | 动作难度、舞台艺术、动作创意、动作流畅、服装造型 |
| 双人赛 | 动作难度、舞台艺术、动作互动、动作创意、服装造型 |
| 团体赛 | 动作难度、舞台艺术、动作互动、动作创意、服装造型 |
| 挑战赛 | 动作难度、动作创意、动作流畅 |

## 🐛 常见问题

**Q: 为什么看不到评分管理标签？**
A: 确保你已登录管理员账号

**Q: 修改评分后大屏幕没有更新？**
A: 大屏幕会在下次刷新时显示新数据

**Q: 可以批量编辑评分吗？**
A: 目前不支持，需要逐条编辑

**Q: 删除的评分可以恢复吗？**
A: 不可以，删除操作是永久的

## 📝 使用建议

1. **谨慎删除**: 删除前确认是否真的需要删除
2. **记录原因**: 建议在修改时记录修改原因
3. **通知相关人员**: 重大修改应通知评委和选手
4. **定期备份**: 建议定期备份数据库
