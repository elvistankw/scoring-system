## 主题功能特性

### 1. 三种主题模式
- **浅色模式**: 明亮清晰，适合白天使用
- **深色模式**: 护眼舒适，适合夜间使用
- **跟随系统**: 自动适配操作系统主题设置

### 2. 持久化存储
- 主题选择保存在 `localStorage` 中
- 键名: `scoring-system-theme`
- 页面刷新后保持用户选择

### 3. 实时预览
- 设置模态框中包含主题预览区域
- 切换主题时立即生效
- 显示当前系统主题（跟随系统模式下）

### 4. 响应式设计
- 模态框适配移动端和桌面端
- 最大高度限制，内容可滚动
- 点击背景关闭模态框

## 技术实现

### 主题提供者
**文件**: `components/shared/theme-provider.tsx`
- 使用 React Context API 管理主题状态
- 监听系统主题变化
- 自动应用主题到 `<html>` 元素的 class

### 样式实现
- 使用 Tailwind CSS 的 `dark:` 前缀
- 所有组件已支持深色模式
- 颜色方案：
  - 浅色: `bg-white`, `text-gray-900`
  - 深色: `dark:bg-gray-800`, `dark:text-white`

## 用户体验

### 交互流程
1. 用户点击右上角用户头像
2. 下拉菜单展开
3. 点击"设置"选项
4. 设置模态框打开
5. 选择主题（浅色/深色/跟随系统）
6. 主题立即生效
7. 点击"关闭"或背景关闭模态框

### 视觉反馈
- 当前选中的主题显示蓝色边框和勾选图标
- 主题切换时有平滑过渡动画
- 按钮悬停时有背景色变化
