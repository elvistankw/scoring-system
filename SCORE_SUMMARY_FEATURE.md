# 评分汇总功能 (Score Summary Feature)

## 功能概述

为评审（Judge）角色添加了评分汇总页面，允许评审查看每个比赛中所有选手的评分详情。

## 新增页面

### 📊 评分汇总页面 (`/[locale]/score-summary`)

**路径**: `app/[locale]/(judge)/score-summary/`

**功能**:
- 选择比赛查看该比赛的所有选手
- 选择选手查看该选手收到的所有评分
- 显示每个评审提交的详细评分数据
- 支持所有比赛类型（个人赛、双人/团队赛、挑战赛）

## 工作流程

### 评审查看评分流程

1. **登录** → 评审登录系统
2. **进入评分汇总** → 从仪表板用户菜单选择"评分汇总"
3. **选择比赛** → 从比赛列表中选择要查看的比赛
4. **选择选手** → 从选手列表中选择要查看评分的选手
5. **查看评分** → 查看该选手收到的所有评分详情

### 评分数据显示

对于每个选手，显示：
- **选手信息**: 姓名、编号、团队名称（如有）
- **评分列表**: 所有评审提交的评分
- **评分详情**: 根据比赛类型显示相应的评分维度

#### 个人赛评分维度
- 动作难度 (action_difficulty)
- 舞台艺术 (stage_artistry)
- 动作创意 (action_creativity)
- 动作流畅 (action_fluency)
- 服装造型 (costume_styling)

#### 双人/团队赛评分维度
- 动作难度 (action_difficulty)
- 舞台艺术 (stage_artistry)
- 动作互动 (action_interaction)
- 动作创意 (action_creativity)
- 服装造型 (costume_styling)

#### 挑战赛评分维度
- 动作难度 (action_difficulty)
- 动作创意 (action_creativity)
- 动作流畅 (action_fluency)

## 特性

### ✅ 用户体验
- **直观的三步流程**: 选择比赛 → 选择选手 → 查看评分
- **清晰的视觉反馈**: 选中状态高亮显示
- **空状态提示**: 友好的空数据提示信息
- **加载状态**: 骨架屏加载动画

### ✅ 响应式设计
- **桌面端**: 双列布局（选手列表 + 评分详情）
- **平板端**: 优化的触摸目标和间距
- **移动端**: 单列堆叠布局

### ✅ 深色模式
- 完全支持浅色/深色主题切换
- 自动适配系统主题设置

### ✅ 性能优化
- **SWR 缓存**: 智能数据缓存和重新验证
- **按需加载**: 只在选择选手时加载评分数据
- **性能监控**: 集成页面加载性能监控

### ✅ 安全性
- **身份验证**: 要求评审登录
- **权限检查**: 验证用户是否具有评审权限
- **Token 验证**: 所有 API 请求使用 JWT 认证

## 权限控制

- **访问权限**: 仅限评审（Judge）角色
- **未登录**: 自动重定向到登录页面
- **非评审用户**: 显示权限错误并重定向

## 错误处理

- **网络错误**: 显示友好的中文错误提示
- **无数据**: 显示空状态提示
- **加载失败**: Toast 通知用户
